### Setup Instruction:

(pay attention to $PATH when setup!)

0. git clone https://github.com/AdrianusWest/fizz_buzz_v2.git
1. cd ./fizz_buzz_v2
2. make build
3. make package-install
4. fizz_buzz_v2
